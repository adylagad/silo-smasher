How Silo Smasher Works

A guided tour of the Autonomous Root-Cause Investigator

①

Metric Monitor ← left panel

Shows 8 weeks of MRR data. The red data points mark the anomaly zone — where the metric diverged from the historical average (dashed line). Key stat cards refresh with real values after each investigation completes.

💡 The pulsing red dot in the incident banner is a live anomaly signal — it stays active until a root cause is confirmed.
②

Ask a Question

Type any business question in plain English — no SQL or dashboards needed. The agent interprets context, generates hypotheses, then tests each one autonomously against real data sources.

💡 Click "MRR drop 15%" in the presets to run the full demo story, then hit Investigate.
③

Investigation Timeline

The agent runs 5 stages sequentially. Each stage lights up in real time as it completes. The connector line between steps turns teal when the step is done.

Neo4j GraphRAG Numeric Finance Tavily News Search Fastino Guardrails Senso Context Yutori Navigator Modulate Voice
④

Hypothesis Cards

Each card shows a hypothesis name, its status, a confidence score, and the evidence used to reach that conclusion. Cards animate in one by one as the agent finishes scoring them.

The Verdict

Final root cause with an overall confidence score, synthesised from graph traversal, finance variance analysis, and external economic signals. The animated percentage counts up as the verdict is revealed.
